# For Loop Practice

## 📖 Project Description

This Python program demonstrates the use of the `for` loop by performing two simple tasks. First, it checks whether each number in a list is even or odd and displays the result. Then, it iterates through a string and prints each letter in uppercase. This project is designed to practice loops, conditional statements, and string manipulation in Python.

## ✨ Features

- Iterates through a list of numbers.
- Determines whether each number is even or odd.
- Prints a message after the loop finishes.
- Iterates through a string character by character.
- Converts each character to uppercase before displaying it.

## 🛠️ Technologies Used

- Python 3

## 📚 Concepts Used

- `for` Loop
- `if` / `else`
- Modulus Operator (`%`)
- Lists
- Strings
- `upper()`
- f-strings
