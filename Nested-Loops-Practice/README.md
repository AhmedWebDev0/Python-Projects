# 🔁 Nested Loops Practice

This project demonstrates how to use nested loops with nested dictionaries in Python.

## 📚 Concepts Covered

- Nested Dictionaries
- Nested Loops
- For Loops
- Dictionary Iteration
- Accessing Nested Data
- Formatted Output using f-strings

## 🎯 Project Description

The program stores people's programming skills and their progress percentages in a nested dictionary. It then uses nested loops to iterate through the data and display each person's skills in a clear and organized format.

## 🚀 Example Output

```text
Skills and Progress For Ahmed Is :
- HTML => 90%
- CSS => 80%
- JavaScript => 90%
```

## 🛠️ Technologies Used

- Python 3

## 📝 Notes

This project was created as part of my Python learning journey to practice working with nested data structures and loops.
