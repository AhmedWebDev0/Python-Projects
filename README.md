# ⚙️ Function Arguments

A simple Python project that demonstrates how to use **functions**, **`*args`**, **`**kwargs`**, and argument unpacking to display a user's programming skills and their proficiency levels.

---

## 📌 Features

* 👤 Accepts the user's name.
* 🛠️ Passes multiple skills using `*args`.
* 📊 Passes skill proficiency using `**kwargs`.
* 📦 Demonstrates tuple unpacking (`*tuple`).
* 📚 Demonstrates dictionary unpacking (`**dictionary`).
* 🖥️ Displays skills and their progress in a clean format.

---

## ▶️ How It Works

1. The user enters their name.
2. A tuple containing programming skills is unpacked using `*`.
3. A dictionary containing skill levels is unpacked using `**`.
4. The function prints the user's name, skills, and proficiency levels.
