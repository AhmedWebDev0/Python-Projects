# 🔁 String Recursion

A simple Python project that demonstrates the power of **recursion** by removing consecutive duplicate characters from a string.

---

## 📌 Features

* 📝 Accepts a string from the user.
* 🔁 Uses recursion instead of loops.
* ✂️ Removes consecutive duplicate characters.
* ⚡ Returns a cleaned version of the string.

---

## 🛠️ Technologies Used

* Python 3

## ▶️ How It Works

1. The user enters a string.
2. The function compares each character with the next one.
3. If two consecutive characters are the same, one of them is removed.
4. The function continues recursively until the entire string is processed.
5. The cleaned string is displayed.
