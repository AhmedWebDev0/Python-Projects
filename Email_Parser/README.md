# 📧 Email Parser

A simple Python project that demonstrates **string slicing** by extracting the username and domain from an email address.

---

## 📌 Features

* 👤 Reads the user's name.
* 📧 Reads an email address.
* ✂️ Extracts the username from the email.
* 🌐 Extracts the email domain.
* 🖥️ Displays the results in a clean format.

---

## 🛠️ Technologies Used

* Python 3

## ▶️ How It Works

Example:

```text
What's your name? Ahmed
What's your Email? ahmed@example.com
```

Output:

```text
Hello Ahmed Your Email Is ahmed@example.com
your user name ahmed
your email example.com
```
